### Prerequisites
The prerequisites are given below
```
*KiCad
```
### Components
```
* 1x  Linear Regulator
* 4x Rectifier diodes
* 1x 470uF capacitor
* 2x Polarized capacitor
* 2x Led 
* 1x 560 ohms resistors 
* 1x Switch DPDT 
* 1x Connector 2x2 odd_even 
```
